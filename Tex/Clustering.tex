and load it up with \href{https://github.com/JuliaIO/JLD.jl}{JLD} as
shown below. (or just load it directly if you have cloned the notebooks)

\begin{itemize}
\item
  Use Affinity Propagraion from
  \href{https://github.com/JuliaStats/Clustering.jl}{Clustering.jl}, to
  cluster word2vec word embeddings, according to meaning.
\item
  Done right this will seperate locations from sports
\item
  Done finely and it will seperate ball-sports from other sports, and
  will seperate locations according to regions, etc
\item
  Affinity propagraion requires a similarity matrix, which you can set
  as a negated distance matrix.
\item
  For this you'll also want
  \href{https://github.com/JuliaStats/Distances.jl}{Distances.jl} for
  all your distance metric needs.
\item
  It is traditional with word2vec to use cosine distance.
\item
  You will as also need to set each item's availability. This is the
  diagonal of the similarity matrix. Decreasing it roughly corresponds
  to decreasing the amount each node wants to be in a cluster on its
  own.
\end{itemize}
